# YieldRadar — Public Roadmap

## v1.1 — Enhanced Risk Scoring (April 2026)

- [x] GoPlus Security API integration (honeypot, malicious contract detection)
- [x] DeFiLlama protocol metadata enrichment (TVL, audit status, protocol tiers)
- [x] APY historical snapshots and trend analytics
- [x] Telegram Stars subscription payments

## v1.0 — Core Product (March 2026)

- [x] Data pipeline with 5-minute polling from DeFiLlama
- [x] Multi-factor risk scoring with 4-tier labels
- [x] Telegram bot with onboarding wizard
- [x] Real-time push alerts
- [x] Daily digest
- [x] Per-user preferences (chains, risk tolerance, filters)
- [x] Freemium tiers (Free: 3/day, Pro: unlimited)
- [x] Docker deployment with health monitoring

## v1.2 — Planned

- [ ] More chains (Optimism, Avalanche, zkSync)
- [ ] Yield comparison within same protocol
- [ ] Portfolio tracking (non-custodial, read-only)
- [ ] Web dashboard
- [ ] Push notifications (Telegram + optional email)

## v2.0 — Future

- [ ] Cross-chain yield arbitrage detection
- [ ] Historical backtesting of yield strategies
- [ ] Community features (shared watchlists, alerts)
- [ ] Mobile app
